# Cellular Forms

An implementation of Andy Lomas' [Cellular Forms](http://www.andylomas.com/extra/andylomas_paper_cellular_forms_aisb50.pdf).

### Dependencies

- [boost](https://www.boost.org/)
- [glfw](https://www.glfw.org/)
- [glm](https://glm.g-truc.net/)

### Usage

    $ make
    $ ./main
